Features

Designed for on-device AI

Atheer combines hardware-accelerated inference with agentic capabilities — all in safe Rust, all running locally on mobile.

⚡

Multi-Backend GPU/NPU

Metal (iOS GPU), Vulkan (Android GPU), NNAPI (Android NPU/DSP), CoreML (ANE) — automatic probe with graceful CPU fallback.

◆

Structured Output

Native grammar-constrained decoding via Pushdown Automaton — guarantees valid JSON, no post-processing needed.

⟳

Agentic Workflows

Built-in tool-calling definitions, extraction, and autonomous agent loops — bring your own tools, let the model decide.

▦

Adaptive Mode Switching

Eco / Balanced / Turbo modes switch in real-time using 1 Hz hardware telemetry — thermal, memory, and battery-aware.

⊞

Memory Hierarchy

L1/L2/L3 KV caching with intelligent eviction policies — maximize throughput within device memory constraints.

◉

Performance-per-Watt

Benchmarked throughput, energy, and thermal throttling curves. Criterion harnesses + perf-bench for CI integration.

Platform Support

iOS & Android, every accelerator

Atheer auto-probes the best available backend for each device. NPU → GPU → CPU, in platform-specific priority order.

iOS — Priority 1

CoreML / ANE

Apple Neural Engine via CoreML compatibility detection. Falls back to Metal for incompatible model architectures.

ANE detection · macOS build required
iOS — Priority 2

Metal GPU

GPU acceleration via Candle Metal backend (Device::Metal). Full tensor ops for prefill and decode.

iOS 15+ · real GPU acceleration
Android — Priority 1

NNAPI NPU/DSP

Android Neural Networks API via raw NDK FFI bindings. Full pipeline: device discovery → model graph → execution.

API 24+ · NDK r29+
Android — Priority 2

Vulkan GPU

Custom GLSL compute shaders (GEMV, flash attention, logits) compiled to SPIR-V via naga at build time.

API 26+ · DP4A-style int8 matmul
Fallback

CPU

Always available fallback using Candle CPU backend. Graceful degradation when no accelerator is available.

Any platform · zero acceleration
Quick Start

From zero to inference

Build the workspace, run tests, and integrate into your iOS or Android app.

bash 
# Build the entire workspace
cargo build --workspace

# Run all ~340 tests
cargo test --workspace

# Download a test model (~350 MB)
scripts/download-test-model.sh

# Run integration tests with a real model
ATHEER_TEST_MODEL=./models/LFM2-700M-Q4_0.gguf \
  cargo test -p atheer-core -- --ignored

# Benchmark throughput
cargo run -p perf-bench -- \
  --model-path model.gguf \
  --batch-sizes 1,4,8
swiftiOS integration
import AtheerFFI

var config = AtheerConfig()
config.modelPath = "/models/llama.gguf"
config.adaptive = true

let engine = AtheerEngine(config: config)
try engine.initialize()

var request = GenerationRequest(prompt: "Hello")
let response = try engine.generateSync(request: request)
print(response.text)
